# User Stories

**Project:** Ayomikun Festus-Olaleye Portfolio
**Version:** 1.0.0
**Date:** 2026-06-28

---

## Persona: Recruiter

**US-001**
As a recruiter, I want to quickly see the candidate's name, role, location, and available status so that I can assess basic fit within 5 seconds of landing.

**US-002**
As a recruiter, I want to download the candidate's CV as a PDF so that I can share it with my hiring team without needing to copy content manually.

**US-003**
As a recruiter, I want to see a clear list of skills organised by category so that I can match the candidate's abilities against an open job description.

**US-004**
As a recruiter, I want to read about professional experience with company names, periods, and specific achievements so that I can evaluate the depth and relevance of the candidate's background.

**US-005**
As a recruiter, I want to click a "Get in touch" button and be taken directly to a contact form so that I can reach out without having to find an email address manually.

## Persona: Hiring Manager / Technical Interviewer

**US-006**
As a technical interviewer, I want to browse the candidate's projects with descriptions, tech stacks, and links to live demos and GitHub repositories so that I can evaluate the quality of their work before an interview.

**US-007**
As a technical interviewer, I want to see which project is featured so that I know which the candidate considers their strongest piece of work.

**US-008**
As a technical interviewer, I want to read expanded project descriptions so that I can prepare specific technical questions about implementation decisions.

## Persona: Potential Collaborator

**US-009**
As a potential collaborator, I want to see what services the candidate offers so that I can determine whether their capabilities align with my project needs.

**US-010**
As a potential collaborator, I want to send a message via a contact form without needing to open my email client so that I can reach out quickly and conveniently.

**US-011**
As a potential collaborator, I want to receive confirmation that my message was sent (or an error if it failed) so that I know whether to follow up via another channel.

## Global / Cross-Cutting

**US-012**
As any visitor, I want to switch the portfolio to my preferred display language (EN / DE / FR / NL / SV) so that I can read the content more comfortably.

**US-013**
As any visitor, I want to toggle between dark and light mode so that the portfolio matches my visual preference or environment.

**US-014**
As any visitor on a mobile device, I want to navigate between sections via a hamburger menu so that I can reach any section without scrolling through the entire page.

**US-015**
As any visitor who has scrolled down, I want a "scroll to top" button so that I can return to the top without manually scrolling.
